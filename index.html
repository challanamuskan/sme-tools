Setup process

From zero to live CRM in 30 minutes

No developer needed after setup. Any Indian SME owner can run this themselves.

1

Fork the repo

One click on GitHub. Your own copy of the CRM — forever free, no lock-in.

2

Edit config.yaml

Change your business name, GSTIN, currency, and toggle features on/off. No code — just a text file.

3

Create free Supabase database

2 minutes. Copy your URL and key. Run the seed SQL to create all tables.

4

Deploy to Streamlit Cloud

Connect your GitHub repo, paste 3 secrets, click Deploy. Live in 60 seconds.
